# Computer Organization & Assembly Language Course Repository
Welcome to the repository for the <b>Computer Organization & Assembly Language</b> course at The University of Alabama at Birmingham (UAB). This repository encapsulates your journey through the intricate world of computer organization and assembly language programming. Through this collection of work, you'll gain insights into the inner workings of computers, the memory hierarchy, compilation processes, C programming principles, makefiles, and the fundamentals of assembly language.

<h3> Course Highlights </h3>
<ul>
  <li> <b>Computer Workings:</b> Gain a comprehensive understanding of how computers operate, from processing information to memory management and allocation. </li>
  <li> <b>Compilation Insights:</b> Delve into the intricate process of compilation, discovering how high-level C code transforms into executable machine code. </li>
  <li> <b>Memory Hierarchy:</b> Explore the different types of memory, their hierarchies, and the principles of memory allocation. </li>
  <li> <b>Programming in C:</b> Master the principles of programming in C, understanding its syntax, data types, and control structures. </li>
  <li> <b>Makefiles:</b> Grasp the significance of makefiles in automating the compilation process and managing complex projects. </li>
  <li> <b>Introduction to Assembly Language:</b> Uncover the basics of assembly language programming, delving into low-level instructions that communicate directly with the hardware. </li>
</ul>

<h3> Topics Explored </h3>
<ul>
  <li>Computer Architecture and Components </li>

<h3> Projects and Assignments </h3>
Throughout the course, you engaged with a variety of projects and assignments that illuminated the concepts explored. These endeavors allowed you to apply your knowledge and gain hands-on experience in various aspects of computer organization and assembly language programming.
